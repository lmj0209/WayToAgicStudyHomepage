# 产品需求文档：个人学术主页 - V8.0 - 侧边栏位置调整（左侧）

## 1. 综述 (Overview)

### 1.1 项目背景

基于 PRD 7.0 的实施反馈，用户发现侧边栏被放置在了右侧，但实际上希望侧边栏在**左侧**，同时保持当前的小间距（gap-3）。本版本专注于**将侧边栏移回左侧**。

### 1.2 核心问题

| 问题 | 当前状态 | 期望状态 |
|------|----------|----------|
| 侧边栏位置 | 右侧 | **左侧** |
| 侧边栏与内容区间距 | gap-3 (12px) | **保持 gap-3** |

### 1.3 参考设计分析

根据 `reference/2.png` 的视觉分析：
- **侧边栏位置**：参考设计的侧边栏位于页面**左侧**
- **内容区位置**：主内容区在右侧
- **间距大小**：侧边栏与内容区之间有适当的间距

## 2. 用户故事详述 (User Stories)

---

### **US-20: 将侧边栏移回左侧，保持小间距**

*   **价值陈述 (Value Statement)**:
    *   **作为** 访问者
    *   **我希望** 侧边栏位于左侧，与内容区保持小间距
    *   **以便于** 符合传统阅读习惯，侧边栏信息在左侧便于查看

*   **业务规则与逻辑**:
    1.  **前置条件**: PRD 7.0 已完成实施，侧边栏在右侧
    2.  **当前布局**:
        ```
        [主内容区 flex-1] ← gap-3 → [侧边栏 lg:w-[280px]]
        ```
    3.  **目标布局**:
        ```
        [侧边栏 lg:w-[280px]] ← gap-3 → [主内容区 flex-1]
        ```
    4.  **实现方式**: 调整 flex 子元素的顺序

*   **验收标准**:
    *   **场景1: 桌面端布局**
        *   **GIVEN** 用户在桌面端浏览
        *   **WHEN** 查看页面布局
        *   **THEN** 侧边栏在左侧，内容区在右侧
    *   **场景2: 间距保持**
        *   **GIVEN** 侧边栏移至左侧
        *   **WHEN** 测量侧边栏与内容区间距
        *   **THEN** 间距为 12px（gap-3）
    *   **场景3: 移动端布局**
        *   **GIVEN** 用户在移动端查看
        *   **WHEN** 浏览页面
        *   **THEN** 移动端侧边栏保持顶部折叠状态

*   **需要修改的文件**:
    ```
    app/page.tsx              # 调整 flex 子元素顺序
    app/academic/page.tsx      # 调整 flex 子元素顺序
    components/Sidebar.tsx     # 调整边框方向
    ```

## 3. 具体修改方案

### 3.1 布局顺序调整

**当前顺序**:
```tsx
<div className="flex flex-col lg:flex-row gap-3">
  <div className="flex-1 min-w-0">
    <Content />
  </div>
  <div className="lg:w-[280px] lg:flex-shrink-0">
    <Sidebar />
  </div>
</div>
```

**修改后顺序**:
```tsx
<div className="flex flex-col lg:flex-row gap-3">
  <div className="lg:w-[280px] lg:flex-shrink-0">
    <Sidebar />
  </div>
  <div className="flex-1 min-w-0">
    <Content />
  </div>
</div>
```

### 3.2 侧边栏边框调整

由于侧边栏移回左侧，边框也需要调整：
- 当前：`border-l`（左边框）
- 修改为：`border-r`（右边框）

### 3.3 布局示意图

```
当前布局：
┌─────────────────────────────────┬──────────────┐
│         Main Content            │   Sidebar    │
│         (flex-1)                │   (右侧)     │
└─────────────────────────────────┴──────────────┘

修改后布局：
┌──────────────┬─────────────────────────────────┐
│   Sidebar    │         Main Content            │
│   (左侧)     │         (flex-1)                │
└──────────────┴─────────────────────────────────┘
       ↑ gap-3 (12px) ↑
```
