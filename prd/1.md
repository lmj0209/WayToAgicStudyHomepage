## 1. 项目概述

### 1.1 项目目标

构建一个**双页面**的个人学术主页，用于展示：
- **页面1：学术展示** - 个人简介、论文列表、学历
- **页面2：AI编程展示** - AI工具推荐、学习笔记博客链接

### 1.2 核心特性

- ✅ 双页面独立设计
- ✅ 中英文双语切换
- ✅ 响应式设计（PC/移动端适配）
- ✅ Vercel 部署 + 后端（动态网页）
- ✅ 配置文件数据输入（JSON/YAML）
- ✅ 简洁学术风格（参考 reference/2.png）
- ❌ 无暗黑模式

### 1.3 MVP定位

- **当前版本**：给自己用，功能完整（学术+AI编程展示）
- **后续版本**：做成生成器工具，给其他科研人员使用

### 3.2 页面1：学术展示

#### 3.2.1 About Me 模块

**内容结构：**

```markdown
## 👋 About Me

[开场白 - 1-2句]
I am a [身份] at [学校], advised by Prof. [导师姓名].

[研究兴趣概述 - 2-3句]
My research focuses on **研究方向1**, **研究方向2**, and **研究方向3**.

[其他信息 - 可选]
Previously, I received my B.S./M.S. degree from [学校] in [年份].
```

**需要您提供：**
- [ ] 导师姓名（中英文）
- [ ] 导师主页链接
- [ ] 研究兴趣描述
- [ ] 本科/硕士学校信息

---

#### 3.2.2 Research Interests 模块

**展示形式：** 标签卡片式

| 研究方向 | 图标建议 |

#### 3.2.3 Publications 模块

**展示形式：** 列表式，按年份倒序排列

**单条论文格式：**

```

会议/期刊名称, 年份
[PDF] [Code] [Slides] [BibTeX]
```

**需要您提供：**
- [ ] 已发表论文列表
- [ ] 每篇论文的链接（PDF、代码、引用等）

#### 3.2.4 Education 模块

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **框架** | Next.js 14 (App Router) | React 生态，支持动态网页 |
| **后端** | Next.js API Routes | 内置后端支持 |
| **样式** | Tailwind CSS | 快速开发，响应式 |
| **多语言** | next-intl | 轻量级 i18n 方案 |
| **图标** | Lucide Icons / React Icons | 开源图标库 |
| **部署** | Vercel | 免费，自动 CI/CD |
| **数据存储** | JSON/YAML 配置文件 | MVP版本简单方案 |
