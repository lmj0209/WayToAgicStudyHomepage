## 1. 项目概述

### 1.1 项目目标

为国防科技大学计算机学院博士生 **黎梦金（Mengjin Li）** 构建一个专业、简洁的个人学术主页，用于展示学术成果、研究方向和个人经历。

### 1.2 核心特性

- ✅ 单页滚动式布局（参考模板二风格）
- ✅ 中英文双语切换
- ✅ 响应式设计（PC/移动端适配）
- ✅ Vercel 部署 + 自定义域名
- ✅ 易于维护更新

## 3. 功能需求详细说明

### 3.1 侧边栏（Sidebar）

| 元素 | 说明 | 必填 |
|------|------|------|
| 头像 | 圆形，建议 200x200px | ✅ |
| 英文名 | Mengjin Li | ✅ |
| 中文名 | 黎梦金 | ✅ |
| 身份 | Ph.D. Student / 博士研究生 | ✅ |
| 学校 | National University of Defense Technology | ✅ |
| 院系 | College of Computer Science | ✅ |
| 地点 | 📍 Changsha, China | ✅ |
| 社交链接 | Email, GitHub, Google Scholar, ORCID | ✅ |

**社交链接配置：**

```yaml
social_links:
  - icon: email
    url: "mailto:your-email@nudt.edu.cn"
  - icon: github
    url: "https://github.com/your-username"
  - icon: google-scholar
    url: "https://scholar.google.com/citations?user=xxx"
  - icon: orcid
    url: "https://orcid.org/0000-0000-0000-0000"
```

### 3.2 About Me 模块

**内容结构：**

```markdown
## 👋 About Me

[开场白 - 1-2句]
I am a Ph.D. student at the College of Computer Science, 
National University of Defense Technology (NUDT), advised by Prof. [导师姓名].

[研究兴趣概述 - 2-3句]
My research focuses on **Computer Architecture**, **Hardware Prefetching**, 
and **Robotics Chips**. I am particularly interested in [具体方向].

[其他信息 - 可选]
Previously, I received my B.S./M.S. degree from [学校] in [年份].
```

**需要您提供：**
- [ ] 导师姓名（中英文）
- [ ] 导师主页链接
- [ ] 更详细的研究兴趣描述
- [ ] 本科/硕士学校信息（如适用）

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **框架** | Next.js 14 (App Router) | React 生态，SSG 支持 |
| **样式** | Tailwind CSS | 快速开发，响应式 |
| **多语言** | next-intl | 轻量级 i18n 方案 |
| **图标** | Lucide Icons / React Icons | 开源图标库 |
| **部署** | Vercel | 免费，自动 CI/CD |
| **域名** | 自定义域名 | Vercel 绑定 |

### 4.2 项目结构

```
academic-homepage/
├── app/
│   ├── [locale]/
│   │   ├── page.tsx          # 主页
│   │   └── layout.tsx
│   └── globals.css
├── components/
│   ├── Sidebar.tsx           # 侧边栏
│   ├── AboutSection.tsx      # About 模块
│   ├── ResearchSection.tsx   # 研究方向
│   ├── PublicationsSection.tsx
│   ├── EducationSection.tsx
│   ├── ExperienceSection.tsx
│   ├── HobbiesSection.tsx
│   ├── LanguageSwitcher.tsx  # 语言切换
│   └── Navigation.tsx
├── messages/
│   ├── en.json               # 英文内容
│   └── zh.json               # 中文内容
├── public/
│   ├── images/
│   │   └── avatar.jpg
│   └── cv.pdf
├── data/
│   └── publications.ts       # 论文数据
├── next.config.js
├── tailwind.config.js
└── package.json
```

## 5. 视觉设计规范

### 5.1 配色方案

| 用途 | 浅色模式 | 深色模式（可选） |
|------|----------|------------------|
| 背景 | `#FFFFFF` | `#1a1a2e` |
| 侧边栏背景 | `#F8F9FA` | `#16213e` |
| 主文字 | `#333333` | `#EAEAEA` |
| 次要文字 | `#666666` | `#A0A0A0` |
| 强调色 | `#2563EB` | `#60A5FA` |
| 链接色 | `#1D4ED8` | `#93C5FD` |

### 5.2 字体

```css
font-family: 
  'Inter', /* 英文 */
  'Noto Sans SC', /* 中文 */
  -apple-system, 
  BlinkMacSystemFont, 
  sans-serif;
```

### 5.3 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| Mobile | < 768px | 单列，侧边栏折叠到顶部 |
| Tablet | 768px - 1024px | 侧边栏收窄 |
| Desktop | > 1024px | 完整双栏布局 |
